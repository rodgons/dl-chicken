# Content Marketing for DownLow Chicken Shack

This project is a static demonstration website created for an ILAC Digital Marketing class activity: **In-Class Task 2: Content Marketing for DownLow Chicken Shack**.

The activity asks students to apply content marketing concepts from a previous website audit by creating an improved sample website for DownLow Chicken Shack. The website is built for academic purposes only and is not affiliated with, published for, or intended to represent the real business.

## Project Objective

The goal is to demonstrate two content marketing formats for a student-focused audience:

- **Evergreen content:** Timeless website content that explains why DownLow Chicken Shack is a strong choice for students, with a focus on value, flavour, and convenience.
- **Ephemeral content:** A temporary, urgency-driven promotion such as a weekly student deal or limited-time combo.

The final website should contain at least two pages and present an improved version of the DownLow Chicken Shack experience based on audit insights and the assigned buyer persona.

## Assignment Requirements

### Evergreen Content

The evergreen section should include:

- A clear headline, such as `Your New Go-To Spot` or `More Than Just Chicken`.
- A short 50-100 word paragraph explaining why DownLow Chicken Shack is a good fit for students.
- A small gallery of 2-3 high-quality chicken sandwich images.
- Content that stays relevant for new customers over time.

### Ephemeral Content

The ephemeral section should include:

- A feature box with a timely headline, such as `This Week's Student Deal!` or `Limited Time: The Finals Fuel Combo`.
- An engaging image and a short description of a fictional weekly special.
- A clear call-to-action button, such as `Order Now` or `Get the Deal`.
- A temporary, urgent feel similar to short-lived social media content.

## Submission Notes

For the class submission, students must publish the website and upload the following to the Moodle dropbox:

- Screenshots of all pages.
- The public website link.

After grading, the website should be deleted or unpublished to avoid confusion for real customers. If the project is kept for a portfolio, screenshots should be saved before deleting or unpublishing the live version.

## Rubric Summary

The project is graded out of 20 points:

- **Evergreen Content:** 8 points
- **Ephemeral Content:** 6 points
- **Persona Alignment:** 4 points
- **Creativity & Presentation:** 2 points

## Local Development

This static demo is built with Next.js. To run it locally:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
