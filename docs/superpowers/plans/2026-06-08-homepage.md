# DownLow Homepage Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the starter Next.js homepage with a static, responsive DownLow Chicken Shack homepage matching the approved design.

**Architecture:** The page is a single static React server component in `src/app/page.tsx`. Styling uses existing Tailwind v4 theme tokens from `src/app/globals.css`, with only small global utilities for grain, glow, and marquee animation.

**Tech Stack:** Next.js 16 App Router, React 19, Tailwind CSS v4, `next/image`, local assets in `public/images`.

---

## File Structure

- Modify `src/app/page.tsx`: Replace starter content with header, hero, ticker, promo, gallery, value cards, FAB, and footer.
- Modify `src/app/globals.css`: Add global `scroll-behavior`, grain overlay utility, text glow utility, marquee keyframes, and reduced-motion fallback.

### Task 1: Global Homepage Utilities

**Files:**
- Modify: `src/app/globals.css`

- [ ] **Step 1: Add global utilities**

Append these rules after the existing `::selection` block:

```css

html {
  scroll-behavior: smooth;
}

.grainy-overlay {
  background-image: radial-gradient(
      circle at 25% 25%,
      rgb(255 255 255 / 0.16) 0 1px,
      transparent 1px 100%
    ),
    radial-gradient(
      circle at 75% 75%,
      rgb(255 255 255 / 0.12) 0 1px,
      transparent 1px 100%
    );
  background-size: 8px 8px, 11px 11px;
  opacity: 0.035;
  pointer-events: none;
}

.text-glow-primary {
  text-shadow: 0 0 20px rgb(255 193 7 / 0.4);
}

.animate-marquee {
  animation: marquee 24s linear infinite;
}

@keyframes marquee {
  from {
    transform: translateX(0);
  }

  to {
    transform: translateX(-50%);
  }
}

@media (prefers-reduced-motion: reduce) {
  html {
    scroll-behavior: auto;
  }

  .animate-marquee {
    animation: none;
  }
}
```
